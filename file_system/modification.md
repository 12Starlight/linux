# **Creating Files and Directories**

| Command  | Description |
|:---------|:------------|
| **`touch<file>`**  | Creates an empty file. e.g. touch ~/Desktop/file1.txt. |
| **`mkdir <directory>`**  | Creates an empty directory. e.g. mkdir ~/newdir. |
